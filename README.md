# Modern Portfolio

A polished personal portfolio starter built with Next.js, TypeScript, and Tailwind CSS.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Production checks

```bash
npm run typecheck
npm run build
```

## Customize next

- Update placeholder identity and links in `data/site.ts`
- Replace `public/resume.pdf` with your resume
- Replace project copy and URLs in `data/site.ts`
- Update `siteConfig.domain` for correct SEO metadata
- Replace `public/og-image.svg` and `public/favicon.svg` if needed
